Introduction

This sample app contains sample to demonstrate how to use Vision API Static Library to make Vision API calls. Specifically:

  • Analyze an image: to get visual features of adult, category, color, image type and faces from an image.
  • Get Thumbnail: to get a small representation of the original image.
  • Optical Character Recognition(OCR): to read text from images in machine-usable character stream.

Build and Run Sample Project

  1. Click Subscription Key Management to obtain your subscription key which is required for Vision API calls.
  2. Open "ProjectOxfordVisionSample\ViewController.m" in xcode.
  3. Replace the empty string of "const kSubscriptionKey" with your subscription key from above step.
  4. Build Project and run.
