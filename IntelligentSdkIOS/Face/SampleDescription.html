Introduction

This sample app contains sample to demonstrate how to use Face API Static Library to make Face API calls.

Build and Run Sample Project

  1. Click Subscription Key Management to obtain your subscription key which is required for Face API calls.
  2. Open "ProjectOxfordFaceSample\AppDelegate.m" in xcode.
  3. Replace the empty string of "const kSubscriptionKey" with your subscription key from above step.
  4. Build Project and run.
